Compétences

Web

Le web est un domaine en pleine expansion, de nombreuses technologies apparaissent permettant de faire de plus en plus de choses simplement avec un navigateur. C'est dans cet ordre d'idée que j'ai décidé d'apprendre les technologies de demain.

BDD

Les bases de données sont la source des contenus. En les interfaçant avec des APIs, on peut récupérer facilement des données.

Logiciel

  • C / C++
  • Java
  • Visual Basic / .NET

Les langages tels que le C sont très utiles lorsqu'il s'agit de toucher au plus bas du système. Ils permettent de gérer la mémoire directement, ce qui en fait un domaine très attractif (systèmes embarqués ou jeu vidéo natif).

OS

Il est important de savoir utiliser un système d'exploitation car il est la source de tout développement informatique ou réseau. Linux est une part importante dans les serveurs et j'attache une grande importance au monde open-source.

Réseaux

  • CCNA (en cours)

La formation Cisco m'est dispensé par des professeurs certifiés. Elle me permet d'acquérir un bon niveau de connaissance en réseau, ce qui n'est pas négligeable pour l'appliquer ou résoudre des problèmes.

Langages de structuration

  • XML
  • JSON (avec des API)
  • YAML

Le stockage et la structuration de données est utile, en particulier lorsqu'il s'agit de manipuler ou de développer des API.

Expériences

Brezeliad : A Tale of Brocéliande

Jeu actuellement en cours de développement. Il a pour but d'être présenté lors du Stunfest 2014. Il est fait en C++ avec la SFML 2.1.
Plus d'informations sur le site du jeu.

SamsarA

Jeu créé en 2014 pendant la Global Game Jam (événement de création de jeux vidéo en 48 heures). Ce jeu est un concept dans lequel le joueur avance à l'aveugle dans sa mémoire et doit se diriger vers certains points. Il a été fait en C++ avec la SFML 2.1.

Heart of Jump

Jeu créé en 2013 pendant la Global Game Jam. Ce jeu mélange le rythme et le "runner". Il a été fait en C++ avec la SFML 2.0.

JeuxAmateurs.fr

Site web sur les jeux vidéo créés par des passionnés sur leur temps libre. Actuellement, le site est gelé mais vous pouvez toujours le visiter ici : old.jeuxamateurs.fr.

8-Bit Robot Music Party

Jeu fait en 2011 pendant le lycée pour un concours organisé par l'école d'ingénieurs EFREI. Il mélange le jeu de plates-formes et de rythme.
Avec ce jeu, nous avons remporté 3 prix : 2nde place dans la catégorie Software, le prix Coup de Coeur and le prix Spécial.
Plus d'informations sur le site du concours.
